# 📈 Daily Investment System

> **MasterWon의 엔지니어 투자자 전략**
> 기계공학 + 자동화 설비 전문성을 레버리지한 한국 시장 투자 시스템

---

## 🎯 Core Philosophy

1. **엣지 없는 영역엔 시간 쓰지 않는다** — 금융/바이오/엔터는 남의 전장
2. **주전장은 산업재·자동화·반도체 장비·2차전지 장비·로봇** — 엔지니어링 엣지 작동 영역
3. **독특함은 "남이 무시하는 영역을 끈질기게 파는 것"** 에서 나온다
4. **Core는 고민 없이, Edge에 시간의 80%, Satellite는 실험**

| 요일 | 작업 | 산출물 |
|------|------|--------|
| 월 | 신규 종목 1개 선정 + 사업보고서 '사업 개요' 섹션 | 사업 1줄 요약 |
| 화 | 재무제표 3개년 (매출/영업이익/순이익) + 부채비율 | 재무 트렌드 메모 |
| 수 | 주요 공시 5건 + 경쟁사 1개 비교 | 경쟁 포지션 메모 |
| 목 | PER / PBR / PSR / ROE + 동종업계 평균 비교 | 밸류에이션 판단 |
| 금 | 3가지 질문 점검 + 투자 결정 (매수/관심/패스) | **결론 노트** |

**3가지 질문 (피셔 프레임):**

1. 이 종목에 대해 시장이 믿는 것 중 **틀린 건** 무엇인가?
2. 남들이 이해 못 하지만 **내(엔지니어)가 이해**할 수 있는 건 무엇인가?
3. 지금 **내 뇌는 어떤 편향**에 빠져있나? (확증편향 / 앵커링 / 손실회피)

### 📊 시장 온도계 — "큰 뉴스 없는 날" 판단법

큰 뉴스 없어도 **조용하다고 방심하면 안 되는 날**이 있다.

| 상황 | 변동성 예상 | 메모 필수 여부 |
|------|-------------|----------------|
| 🟢 뉴스 無 + 수급 안정 + 환율 ±5원 | ±0.5% | 간단히만 |
| 🟡 뉴스 無 + 외국인 대량 매도 | ±1~2% | 수급 분석 필수 |
| 🟡 월/분기말 리밸런싱 시즌 | ±1~2% | 섹터 로테이션 주목 |
| 🟡 옵션 만기일 (매월 2째 목) | ±1~3% | 변동성 확대 가능 |
| 🔴 미국 선물 -1% 이상 + 아시아 동반 약세 | -1~3% | 방어 모드 |
| 🔴 환율 10원 이상 급등 | 섹터 양극화 | 수출주/내수주 체크 |

**결론:** "뉴스 없으니 조용하겠지" = **가장 위험한 생각**
→ 수급, 환율, 미국 선물, 만기일을 매일 체크해야 진짜 변동성 예측 가능

## 🎯 Edge 종목 선정 기준

### ✅ MasterWon 엣지 작동 영역

- **반도체 장비** (한미반도체, 원익IPS, 주성엔지니어링 등)
- **2차전지 장비** (에코프로에이치엔, 피엔티, 필옵틱스 등)
- **자동화 / 로봇** (로보스타, 라온피플, 뉴로메카 등)
- **공장 자동화 / 머신비전** (고영, 이오테크닉스 등)
- **기계 / 산업재** (두산에너빌리티 계열, 현대위아 등)

### ❌ 엣지 없는 영역 (피할 것)

- 바이오 / 제약
- 은행 / 보험 / 증권
- 엔터 / 게임 / 콘텐츠
- 리츠 / 부동산

### 📋 종목 분석 체크리스트

- [ ] 이 기업이 **실제로 만드는 것**이 뭔지 엔지니어 관점에서 이해되는가?
- [ ] 기술 해자가 존재하는가? (특허 / 양산 노하우 / 고객 락인)
- [ ] 주요 고객사가 누구인가? (삼성/SK/LG/글로벌 Tier 1?)
- [ ] 경쟁사 대비 **기술적 차별점**은? (본인이 판단 가능)
- [ ] 3개년 매출 성장률 + 영업이익률 추이
- [ ] PER / PBR / PSR 동종업계 평균 대비 위치
- [ ] 오너 / 대주주 지분율 + 최근 매매 내역
- [ ] 최근 대규모 CAPEX 공시 여부 (장비 투자 = 수주 신호일 가능성)
